Avoda Zara daf 62a (Perek 5 - Michna 1)

Le salarié travaillant avec du vin destiné à un sacrifice

  • ⚡ Salaire interdit — Si le salarié travaillait avec du vin pour un sacrifice, son salaire est interdit et il est interdit d’en bénéficier.
  • ✅ Salaire permis — Si le salarié effectuait un autre travail, même si on lui a demandé de déplacer un tonneau de vin de sacrifice, son salaire est permis.

Loueur du âne pour transporter du vin de sacrifice

  • ⚡ Location interdite — Si l’âne est loué pour porter du vin de sacrifice, le montant de la location est interdit.
  • ✅ Location permise — Si l’âne est loué pour s’asseoir dessus, même si un tonneau de vin de sacrifice a été posé dessus, le montant de la location est permis.

Avoda Zara daf 62a (Perek 5 - Michna 1) - Salaire et location liés au vin de sacrifice : quand le bénéfice est interdit ou permis

Explication de la Michna : Salaire et location liés au vin de sacrifice

  • ⚡ Principe général : Tout bénéfice provenant d’un objet ou d’une action directement liée à un sacrifice idolâtre (vin de nesekh) est interdit.
  • ✅ Exception : Si l’action ou l’usage n’est pas directement destiné au sacrifice idolâtre, le bénéfice peut être permis.

1. Le salarié travaillant avec du vin de sacrifice

  • ⚡ Salaire interdit : Si le salarié effectue un travail directement sur du vin utilisé pour un sacrifice, son salaire est interdit. Le Juif ne peut pas en bénéficier car le salaire provient d’une activité idolâtre.
  • ✅ Salaire permis : Si le salarié effectue un travail autre que pour le vin de sacrifice, même si on lui demande ponctuellement de déplacer un tonneau de vin de sacrifice, le salaire est permis. Ici, l’activité principale n’est pas idolâtre, donc le bénéfice est autorisé.

2. Louer un âne pour le transport du vin de sacrifice

  • ⚡ Location interdite : Si l’âne est loué spécifiquement pour transporter du vin de sacrifice, le paiement de la location est interdit. Le revenu est directement lié à l’adoration idolâtre.
  • ✅ Location permise : Si l’âne est loué pour s’asseoir dessus ou pour un usage autre que le transport du vin de sacrifice, le paiement est permis, même si un tonneau de vin de sacrifice est accidentellement posé dessus. L’usage principal n’est pas idolâtre.
