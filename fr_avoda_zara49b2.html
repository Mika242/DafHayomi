Avoda Zara daf 49b (Perek 3 - Michna 15)

Comment annuler le statut idolâtre d’un ashera ?

  • 🌿 Bois coupé ou branches prélevées : Si un non‑juif prend un bâton, une baguette ou même une feuille de l’ashera, le statut idolâtre de l’arbre est annulé, car cela montre qu’il ne l’adore plus.
  • ✂️ Taille de l’arbre pour son propre usage : Si le non‑juif taille l’arbre pour son usage personnel, il reste interdit d’en tirer profit.
  • 🪓 Taille de l’arbre non pour son usage personnel : Si l’arbre est taillé sans besoin personnel, il devient permis d’en tirer profit.

Mishna :

  • 🌿 Si le non‑juif a coupé ou émondé l’arbre, pris un bâton ou une feuille — le statut idolâtre est annulé.
  • ✂️ S’il a taillé l’arbre pour son usage personnel — il est interdit d’en profiter.
  • 🪓 S’il a taillé l’arbre sans usage personnel — il est permis d’en profiter.

Avoda Zara daf 49b (Perek 3 - Michna 15) - Annulation du statut idolâtre d’un ashera : règles sur la coupe, l’usage et le profit

Explication de la Michna : Annulation du statut idolâtre d’un ashera

  • 🌿 Principe général : Le statut idolâtre d’un arbre (ashera) peut être annulé si un non‑juif démontre qu’il n’en use plus pour l’idolâtrie.
  • ✂️ Cas de la coupe ou émondage :
    • Si un non‑juif coupe du bois sec ou émonde du bois vert, ou prélève un bâton, une baguette ou même une feuille, le statut idolâtre est annulé.
    • Raison : Le non‑juif utilise l’arbre pour son propre profit et non pour l’adoration, prouvant que l’objet n’est plus sacré.
  • 🪓 Cas de la taille pour usage personnel :
    • Si le non‑juif taille l’arbre pour améliorer son apparence ou pour un usage personnel (par exemple, pour fabriquer quelque chose ou embellir l’arbre), le statut idolâtre reste en vigueur.
    • Interdiction : On ne peut toujours pas tirer profit de l’arbre tant qu’il est utilisé pour l’idolâtrie ou son usage personnel justifie son interdiction.
  • ✅ Cas de la taille non liée à un usage personnel :
    • Si l’arbre est taillé ou modifié par le non‑juif mais pas pour son usage propre, le statut idolâtre est annulé et il devient permis d’en tirer profit.

Tableau de synthèse :

Action du non‑juif Statut de l’ashera Possibilité d’en profiter
Prélève bois, bâton, baguette ou feuille Annulé ✅ Permis
Taillé pour usage personnel Maintenu ❌ Interdit
Taillé sans usage personnel Annulé ✅ Permis
